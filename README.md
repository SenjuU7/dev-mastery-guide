## ✨ Fitur

- **28 bab lengkap** yang mencakup tiga topik utama
- **Sidebar navigasi** dengan progress tracker per section
- **5 tab per bab**: Konsep · Contoh · Real Case · Kesalahan Umum · Latihan
- **Validasi progress** — bab ditandai selesai hanya setelah kamu klik "Tandai Selesai"
- **Dark mode** penuh secara default
- **Responsive** — sidebar collapse di layar mobile
- **Zero dependency** — tidak ada npm, tidak ada CDN wajib (font Google Fonts opsional)
- **Syntax highlighting** manual untuk Python, SQL, dan Rust

## 🚀 Cara Pakai

1. **Clone repo ini**
   ```bash
   git clone https://github.com/username/dev-mastery-guide.git
   cd dev-mastery-guide
   ```

2. **Buka di browser**
   ```bash
   # macOS
   open data_mastery_guide.html

   # Linux
   xdg-open data_mastery_guide.html

   # Windows
   start data_mastery_guide.html
   ```
   Atau cukup drag & drop file ke browser.

3. **Mulai belajar!**
   - Klik bab di sidebar kiri
   - Baca semua tab (Konsep → Contoh → Real Case → dst.)
   - Klik **"✓ Tandai Selesai"** di akhir bab
   - Pantau progress di bagian bawah sidebar
